As 20 tags do HTML

<html>

A tag <html> serve para definir que o documento é HTML.

Exemplo:

<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Meu Primeiro Site</title>
</head>
<body>
<h1>As 20 tags do HTML</h1>
<h2><html></h2>
<p>A tag <html> serve para definir que o documento é HTML.</p>
</body>
</html>

*Importante: Todas as tags de um código HTML devem ter começo e fim. Isso é, após começar com <comando>, é necessário terminar com </comando> quando concluir.

<head>

A tag <head> contém os metadados do site.

Exemplo:

<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Meu Primeiro Site</title>
</head>

<title>

A tag <title> define o título da aba do navegador.

Exemplo:

<title>Meu Primeiro Site</title>

<body>

A tag <body> define todo o conteúdo visível para o usuário.

Exemplo:

<body>
<h1>As 20 tags do HTML</h1>
<h2><html></h2>
<p>A tag <html> serve para definir que o documento é HTML.</p>
</body>

<meta>

A tag <meta> fornece informações sobre o documento, como o formato de codificação.

Exemplo:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<div>

A tag <div> serve para criar uma linha nova sem pular parágrafos. Toma todo o espaço horizontal da janela do navegador. Um texto comicamente grande usando esse formato só vai quebrar a linha no limite da página.

Exemplo:

<div>Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor [...]</div>

<span>

A tag <span> serve para modificar um trecho específico dentro de um texto.

Exemplo:

<div>Eu sou uma <strong>DIV (Bloco 1)</strong></div>
<div>Eu sou outra <strong>DIV (Bloco 2)</strong></div>

<p>
Isto é um parágrafo onde
<span><strong>esta parte é um span</strong></span>
e continua na mesma linha.
</p>

--------------------------------------------------------------------------------------------

Eu sou uma DIV (Bloco 1)
Eu sou outra DIV (Bloco 2)

Isto é um parágrafo onde esta parte é um span e continua na mesma linha.

As tags <h>

As tags que vão de <h1> até <h6> - formalmente conhecidas como cabeçalhos hierárquicos - classificam diferentes tamanhos de título.

Exemplo:

<h1>Título h1</h1>
<h2>Título h2</h2>
<h3>Título h3</h3>
<h4>Título h4</h4>
<h5>Título h5</h5>
<h6>Título h6</h6>

--------------------------------------------------------------------------------------------

Título h1

Título h2

Título h3

Título h4

Título h5
Título h6

<p>

A tag <p> começa um novo parágrafo - como este que você está lendo.

Exemplo:

<p>A tag <p> começa um novo parágrafo - como este que você está lendo.</p>

<ul>

A tag <ul> ordena itens de forma não numerada.

<ol>

A tag <ol> ordena itens de forma numerada.

<li>

A tag <li> define cada item dentro de uma lista (<ul> ou <ol>).

Exemplo:

<ul>
<li>Banana</li>
<li>Maçã</li>
<li>Laranja</li>
<li>Uva<li>
</ul>

<ol>
<li>Primeiro</li>
<li>Segundo</li>
<li>Terceiro</li>
<li>Quarto</li>
</ol>

--------------------------------------------------------------------------------------------

  • Banana
  • Maçã
  • Laranja
  • Uva
  1. Primeiro
  2. Segundo
  3. Terceiro
  4. Quarto

<br>

A tag <br> força uma quebra de linha simples.

Exemplo:

<p>
<div>Eu posso começar uma frase aqui em cima...<div>
<br>
<div>...e terminar aqui embaixo pulando uma linha.<div>
</p>

--------------------------------------------------------------------------------------------

...e terminar aqui embaixo pulando uma linha.
