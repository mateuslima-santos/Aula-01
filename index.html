As 20 tags do HTML

<html>

A tag <html> serve para definir que o documento é HTML.

Exemplo:

<html lang="en">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Meu Primeiro Site</title>

</head>

<body>

<h1>As 20 tags do HTML</h1>

<h2><html></h2>

<p>A tag <html> serve para definir que o documento é HTML.</p>

</body>

</html>

*Importante: Todas as tags de um código HTML devem ter começo e fim. Isso é, após começar com <comando>, é necessário terminar com </comando> quando concluir.

<head>

A tag <head> contém os metadados do site.

Exemplo:

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Meu Primeiro Site</title>

</head>

<title>

A tag <title> define o título da aba do navegador.

Exemplo:

<title>Meu Primeiro Site</title>

<body>

A tag <body> define todo o conteúdo visível para o usuário.

Exemplo:

<body>

<h1>As 20 tags do HTML</h1>

<h2><html></h2>

<p>A tag <html> serve para definir que o documento é HTML.</p>

</body>
